# Dataset de Rendimiento Estudiantil (Student Performance Dataset)

## Descripción

El **Student Performance Dataset** es un conjunto de datos diseñado para analizar los factores que influyen en el rendimiento académico de los estudiantes. Contiene **10,000 registros de estudiantes**, donde cada registro incluye información sobre diversas variables predictoras y un índice de desempeño académico.

## Variables Predictoras

### 1. Hours Studied
Cantidad total de horas dedicadas al estudio por cada estudiante.

### 2. Previous Scores
Calificaciones obtenidas por los estudiantes en evaluaciones anteriores.

### 3. Extracurricular Activities
Indica si el estudiante participa en actividades extracurriculares.

- **Yes**: Participa.
- **No**: No participa.

### 4. Sleep Hours
Promedio de horas de sueño que el estudiante tiene por día.

### 5. Sample Question Papers Practiced
Número de exámenes o cuestionarios de práctica realizados por el estudiante.

---

## Variable Objetivo (Target)

### Performance Index
Medida del rendimiento académico general del estudiante.

**Características:**
- Representa el desempeño académico global.
- Está redondeado al entero más cercano.
- Toma valores entre **10 y 100**.
- Un valor más alto indica un mejor rendimiento académico.

---

## Objetivo del Dataset

Este conjunto de datos busca proporcionar información sobre la relación entre las variables predictoras y el índice de desempeño académico. Puede ser utilizado por investigadores y analistas de datos para estudiar cómo influyen factores como:

- Las horas de estudio.
- Las calificaciones previas.
- La participación en actividades extracurriculares.
- Las horas de sueño.
- La práctica mediante exámenes de muestra.

en el rendimiento académico de los estudiantes.

---

## Nota Importante

> **P.S.** Este dataset es **sintético** y fue creado con fines ilustrativos. Por lo tanto, las relaciones observadas entre las variables y el índice de desempeño pueden no reflejar escenarios reales.
