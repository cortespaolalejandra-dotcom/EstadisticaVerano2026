# 🎓 Student Performance Dataset

## 📖 Descripción General

El **Student Performance Dataset** es un conjunto de datos diseñado para estudiar los factores que influyen en el rendimiento académico de los estudiantes. Contiene **10,000 registros**, donde cada observación representa a un estudiante y recopila información sobre diferentes características personales y académicas, así como una medida de su desempeño final.

Este dataset permite analizar cómo variables relacionadas con los hábitos de estudio, el descanso y la experiencia académica previa pueden estar asociadas con el rendimiento escolar.

---

## 📊 Variables del Conjunto de Datos

### ⏱️ Hours Studied

Número total de horas dedicadas al estudio por cada estudiante.

### 📝 Previous Scores

Calificaciones obtenidas por el estudiante en evaluaciones anteriores.

### 🎭 Extracurricular Activities

Indica si el estudiante participa en actividades extracurriculares.

| Valor | Descripción                                   |
| ----- | --------------------------------------------- |
| Yes   | Participa en actividades extracurriculares    |
| No    | No participa en actividades extracurriculares |

### 😴 Sleep Hours

Promedio de horas de sueño por día.

### 📚 Sample Question Papers Practiced

Cantidad de exámenes o cuestionarios de práctica realizados por el estudiante.

---

## 🎯 Variable Objetivo

### Performance Index

El **Performance Index** representa el nivel de desempeño académico general de cada estudiante.

**Características principales:**

* 📈 Rango de valores: **10 a 100**
* 🔢 Redondeado al entero más cercano
* 🏆 Valores más altos indican un mejor rendimiento académico

---

## 🔍 Propósito del Dataset

Este conjunto de datos permite explorar la relación entre diferentes factores académicos y personales con el desempeño estudiantil. Entre las preguntas que pueden investigarse se encuentran:

* ¿Cómo influyen las horas de estudio en el rendimiento académico?
* ¿Existe relación entre las calificaciones previas y el desempeño actual?
* ¿La participación en actividades extracurriculares afecta el rendimiento?
* ¿Qué impacto tienen las horas de sueño?
* ¿La práctica mediante exámenes de muestra mejora los resultados académicos?

Estas preguntas convierten al dataset en una excelente herramienta para ejercicios de análisis exploratorio, visualización de datos y modelos predictivos.

---

## ⚠️ Consideraciones

> **Importante:** Este conjunto de datos es completamente **sintético** y fue creado con fines educativos e ilustrativos. Por lo tanto, las relaciones observadas entre las variables pueden no reflejar comportamientos reales de estudiantes.
