# 🏋️ Gym Members Exercise Dataset

### 📊 Análisis de rutinas de ejercicio, atributos físicos y métricas de condición física

</div>

---

## 📖 Descripción

Este conjunto de datos proporciona una visión detallada de las **rutinas de ejercicio**, los **atributos físicos** y las **métricas de condición física** de los miembros de un gimnasio.

Contiene **973 registros**, cada uno con información sobre el rendimiento durante el entrenamiento, características demográficas y nivel de experiencia de los usuarios, lo que permite realizar análisis estadísticos, modelos predictivos y estudios sobre la salud y el desempeño físico.

---

## 📋 Variables del conjunto de datos

| 🏷️ Variable | 📝 Descripción |
|:------------|:---------------|
| 👤 **Age** | Edad del miembro del gimnasio. |
| 🚻 **Gender** | Género del miembro (*Male* o *Female*). |
| ⚖️ **Weight (kg)** | Peso en kilogramos. |
| 📏 **Height (m)** | Estatura en metros. |
| ❤️ **Max_BPM** | Frecuencia cardíaca máxima durante el entrenamiento. |
| 💓 **Avg_BPM** | Frecuencia cardíaca promedio durante la sesión. |
| 🫀 **Resting_BPM** | Frecuencia cardíaca en reposo antes del entrenamiento. |
| ⏱️ **Session_Duration (hours)** | Duración de la sesión de entrenamiento (horas). |
| 🔥 **Calories_Burned** | Calorías quemadas durante la sesión. |
| 🏃 **Workout_Type** | Tipo de entrenamiento (*Cardio, Strength, Yoga, HIIT*). |
| 📉 **Fat_Percentage** | Porcentaje de grasa corporal. |
| 💧 **Water_Intake (liters)** | Consumo diario de agua durante el entrenamiento. |
| 📅 **Workout_Frequency (days/week)** | Número de entrenamientos por semana. |
| 🎯 **Experience_Level** | Nivel de experiencia (1 = Principiante, 2 = Intermedio, 3 = Experto). |
| 📐 **BMI** | Índice de Masa Corporal (IMC). |

---

## 💡 ¿Para qué sirve este conjunto de datos?

Este dataset es ideal para realizar:

- 📈 Análisis exploratorio de datos (EDA).
- 🤖 Modelos de aprendizaje automático.
- 🏋️ Estudios sobre hábitos de entrenamiento.
- ❤️ Investigaciones relacionadas con la salud y el rendimiento físico.
- 📊 Análisis de la relación entre variables demográficas y fisiológicas.
- 🔍 Identificación de factores que influyen en las calorías quemadas, la frecuencia cardíaca y la intensidad del ejercicio.

---

## ⭐ Resumen

> Este conjunto de datos ofrece una base sólida para comprender cómo las características físicas, los hábitos de entrenamiento y la experiencia de los usuarios influyen en su rendimiento y condición física. Gracias a la diversidad de variables disponibles, es una excelente opción para proyectos de análisis de datos, estadística y ciencia de datos.
